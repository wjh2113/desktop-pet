# 桌面萌宠

一个 Windows 桌面萌宠，使用 Python + Tkinter 编写，支持置顶、拖动、动态情绪、Q 弹下落、番茄钟、语音对话、活动统计、今日三件事、久坐护眼提醒、每日总结和可选云同步。

## 运行

双击 `start-pet.bat`，或者运行：

```powershell
cd E:\openclaw\workspace\projects\desktop-pet
python pet.py
```

## 操作

- 鼠标左键拖动萌宠。
- 双击切换心情；它也会根据专注、完成任务、久坐提醒、投喂、玩耍等状态自动变换情绪和表情。
- 萌宠会定期主动冒出一句鼓励、问候或小互动；专注时会降低频率，不会一直打扰。
- 右键 `切换形象` 可以选择猫猫、小兔子、小熊、小老虎。
- 萌宠会自动切换喝水、劳动、植树、过家家、小老师、学习、厨师、画画、小医生等动作/角色；部分角色会带场景、服装、道具和动作，也可以在对话里直接说“老师”“学习”“喝水”“植树”等命令来指定。
- 右键打开中文菜单。
- 选择 `Q弹下落`，或让萌宠获得焦点后按 `Space`，可以从屏幕上方掉下来并带 Q 弹音效。
- 右键菜单勾选 `番茄钟` 后，萌宠旁边会显示一个玻璃态迷你番茄钟，可以直接开始、暂停和重置；取消勾选即可隐藏。
- 选择 `和萌宠对话`，可以打字聊天，也可以点 `语音输入/命令` 直接说话；普通话会聊天，像“开始番茄钟”“打开统计”“切换小老虎”“Q弹下落”这样的指令会自动执行。
- 右键 `DeepSeek 配置` 可以填写 API Key、模型和接口地址；配置后普通聊天会调用 DeepSeek，命令仍优先在本地执行。
- 右键可以 `查天气`、`标记情绪`、配置 `名字/偏好/性格`、查看 `成就打卡`，也可以勾选 `勿扰模式`。
- 选择 `显示/隐藏小面板`，在萌宠旁边显示悬浮小面板，快速查看番茄钟、三件事和当前应用。
- 选择 `今日看板`，打开整合后的今日页面，可以同时查看时间统计、应用耗时图、窗口明细榜、今日三件事和萌宠总结建议。
- 选择 `今日三件事`，记录今天最重要的三件事并勾选完成状态。
- 在 `今日看板` 里点击 `生成 Markdown 总结`，生成当天 Markdown 总结报告。
- 右键 `云同步` 可以登录/配置云端地址，或立即同步本机记录。
- 萌宠获得焦点时按 `Esc` 可以退出。

## 时间管理功能

- 今日三件事保存在 `data/tasks-YYYY-MM-DD.json`。
- 活动窗口统计保存在 `data/activity-YYYY-MM-DD.json`。
- 每日总结报告保存在 `reports/YYYY-MM-DD.md`。
- 久坐/护眼提醒会在连续使用一段时间后自动播报。
- 专注模式会在番茄钟专注中检测明显分心窗口，并轻声提醒回到当前任务。

默认情况下，所有数据都只保存在本机。配置云同步后，`data/*.json` 和 `reports/*.md` 会上传到你指定的云端服务，并在多台电脑之间共享。

## 云同步

云同步使用项目内置的轻量 HTTP 服务，不依赖第三方数据库或 SDK。你可以在一台长期在线的电脑、NAS 或云服务器上运行：

```powershell
cd E:\openclaw\workspace\projects\desktop-pet
python scripts\cloud_server.py
```

服务默认监听 `http://0.0.0.0:8765`。如需修改地址或端口：

```powershell
$env:PET_CLOUD_HOST="0.0.0.0"
$env:PET_CLOUD_PORT="8765"
python scripts\cloud_server.py
```

然后在每台电脑的萌宠右键菜单里打开 `云同步` -> `登录/配置`：

- 云端地址填写 `http://服务器IP:8765`。
- 多台电脑使用同一个账号和密码即可同步同一份数据。
- 首次登录会自动创建账号，之后使用相同密码登录。
- 云端数据保存在 `cloud-data/`，本机登录配置保存在 `data/cloud-config.json`。

## 语音对话

语音输入使用 Windows 自带的 `System.Speech` 识别器和默认麦克风。
如果系统没有安装中文语音识别器，可能会识别不准或提示没有可用识别器；文字聊天和语音回复仍然可用。

## DeepSeek 对话

- DeepSeek 配置保存在 `data/deepseek-config.json`，只保存在本机，不会上传到云同步，也不会提交到 Git。
- 默认接口地址是 `https://api.deepseek.com`，默认模型是 `deepseek-v4-flash`。
- 没有配置 API Key 时，萌宠会继续使用本地规则回复。
- 已配置 API Key 时，普通聊天会调用 DeepSeek；“开始番茄钟”“老师”“植树”“打开统计”等命令仍然优先本地执行。
- 系统提示词会根据当前场景、主人情绪、个人偏好和角色性格包自动变化，例如番茄钟专注、小老师、植树、过家家、睡觉、小医生、久坐拉伸等。

## 陪伴功能

- 天气：右键 `查天气` 会先尝试按 IP 定位城市，再调用天气接口。可在 `名字/偏好/性格` 里填写固定城市和墨迹天气 URL 模板，模板支持 `{city}`。
- 情绪：右键 `标记情绪` 后，萌宠会切换相应语气和陪伴场景，并记录到本地。
- 偏好：`data/pet-config.json` 保存萌宠名字、主人称呼、性格包、天气城市和 DeepSeek 个人偏好。
- 成就：`data/companion-state.json` 保存情绪打卡、首次天气、首次番茄钟、三件事全完成等本地成就。
- 勿扰：勾选 `勿扰模式` 后，主动互动、分心提醒、久坐提醒会安静下来。
- 每日回顾：配置 DeepSeek 后，生成 Markdown 总结时会追加一段智能回顾。
