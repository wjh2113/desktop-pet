# 桌面萌宠

一个 Windows 桌面萌宠，使用 Python + Tkinter 编写，支持置顶、拖动、Q 弹下落、番茄钟、语音对话、活动统计、今日三件事、久坐护眼提醒和每日总结。

## 运行

双击 `start-pet.bat`，或者运行：

```powershell
cd E:\openclaw\workspace\projects\desktop-pet
python pet.py
```

## 操作

- 鼠标左键拖动萌宠。
- 双击切换心情。
- 右键打开中文菜单。
- 选择 `Q弹下落`，或让萌宠获得焦点后按 `Space`，可以从屏幕上方掉下来并带 Q 弹音效。
- 选择 `开始番茄钟`，进入 25 分钟专注 + 5 分钟休息循环。
- 选择 `和萌宠对话`，可以打字聊天，也可以点 `语音输入` 直接说话，萌宠会语音回复。
- 选择 `今日时间统计`，打开带数字卡片、应用耗时图和窗口明细榜的时间看板。
- 选择 `今日三件事`，记录今天最重要的三件事并勾选完成状态。
- 选择 `生成今日总结`，生成当天 Markdown 总结报告。
- 萌宠获得焦点时按 `Esc` 可以退出。

## 时间管理功能

- 今日三件事保存在 `data/tasks-YYYY-MM-DD.json`。
- 活动窗口统计保存在 `data/activity-YYYY-MM-DD.json`。
- 每日总结报告保存在 `reports/YYYY-MM-DD.md`。
- 久坐/护眼提醒会在连续使用一段时间后自动播报。

所有数据都只保存在本机，不会发送到外部。

## 语音对话

语音输入使用 Windows 自带的 `System.Speech` 识别器和默认麦克风。
如果系统没有安装中文语音识别器，可能会识别不准或提示没有可用识别器；文字聊天和语音回复仍然可用。
